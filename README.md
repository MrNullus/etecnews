<h1>ETECNews</h1>

<p>📰 Um projeto de notícias desenvolvido com o Laravel</p>

## 📚 Descrição

O ETECNews é um projeto de um site de notícias desenvolvido com o framework Laravel. Ele permite que os usuários visualizem e leiam as últimas notícias, cadastrem-se para receber atualizações e interajam com o conteúdo através de comentários.

## ⚙️ Funcionalidades

- Exibição de notícias em destaque na página inicial
- Listagem de todas as notícias por categoria
- Página de detalhes de cada notícia
- Possibilidade de fazer comentários nas notícias
- Cadastro de usuários para receber atualizações por e-mail

## 📁 Estrutura de Arquivos

- `app/`: Contém a lógica de negócio do projeto
- `public/`: Pasta raiz para arquivos públicos (CSS, JavaScript, imagens)
- `resources/`: Contém as visualizações, arquivos de tradução e outros recursos
- `routes/`: Define as rotas do projeto
- `database/`: Arquivos de migração e sementes do banco de dados
- `tests/`: Testes automatizados para o projeto

## 🚀 Instalação

1. Clone este repositório em sua máquina local.
2. Navegue até o diretório do projeto.
3. Execute o comando `composer install` para instalar as dependências do Laravel.
4. Faça uma cópia do arquivo `.env.example` e renomeie-o para `.env`.
5. Configure as informações do banco de dados no arquivo `.env`.
6. Execute o comando `php artisan key:generate` para gerar uma chave de criptografia única.
7. Execute o comando `php artisan migrate` para executar as migrações do banco de dados.
8. Execute o comando `php artisan serve` para iniciar o servidor de desenvolvimento.
